# BMI Calculator

A simple BMI Calculator created using Python and Jupyter Notebook.

## Features
- Takes user weight and height as input
- Calculates BMI
- Classifies BMI into categories:
  - Underweight
  - Normal Weight
  - Overweight
  - Obese

## Technologies Used
- Python
- Jupyter Notebook

## How to Run
1. Open the notebook in Jupyter Notebook.
2. Run all cells.
3. Enter your weight and height when prompted.
4. View your BMI result.
